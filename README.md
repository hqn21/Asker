# Asker
English | [繁體中文](docs/README_zh-tw.md)
## About The Project
This project is a online message box which allows users to create their own account and receive messages from anonymous visitors.
### Requirements
* Apache
* PHP 7.2
* JavaScript ES6
### Built With
* MySQL
* jQuery
* Bootstrap
